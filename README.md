# ConCaVa GUI

> Graphical interface for managing ConCaVa metadata.

## Usage

```bash
git clone https://github.com/kukua/concava-gui.git
cd concava-gui
npm install
grunt build
npm start
# > Navigate to localhost:8080

# Developing
grunt watch
npm start
```
